Features

  • User friendly

    With a Windows 2000-style wizard-based user interface, end users will find FreeExtractor-created archives intuitive, fast and easy to use. Self-extractor authors follow a simple 4-step wizard which guides them through the process.
     

  • Free and Open Source

    FreeExtractor is free for all uses, and is released under a zLib-style license. It is unencumbered by restrictive licensing agreements. You are free to install and redistribute an unlimited number of royalty-free self-extractors created by FreeExtractor. Developers have the added flexibility and freedom of modifying the source code to suit their application.
     
    If you find FreeExtractor useful, you are encouraged to make a small donation to help further development.
     

  • Customizable

    Authors have fine-grained control over archive settings and can tailor their self-extractors to suit their application. Authors can specify such options as an embedded URL, introduction text, shortcut creation and a custom icon.

    In addition, the self-extractor creation process can be scripted and automated using MakeSFX.exe - the command-line version of FreeExtractor.
     

  • Compatible

    FreeExtractor runs on all 32-bit versions of Windows (Windows 95/98/ME/NT/2000). Advanced end-users have the flexibility of extracting archive contents with third party programs like WinZip and WinRAR.
     

  • Compact

    As tested, FreeExtractor produces the smallest self-extracting archives of any product - up to 10kb smaller than any competing product. In addition, the FreeExtractor Wizard (used to create self extracting archives) is only 38kb in size - the smallest SFX generator we are aware of.
