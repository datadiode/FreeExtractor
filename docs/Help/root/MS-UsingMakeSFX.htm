Using MakeSFX

MakeSFX.exe is the optional, command-line version of FreeExtractor. MakeSFX was designed specifically for software developers who wish to automate the process of packaging their software (e.g. through build scripts). Its output is identical to that of the FreeExtractor Wizard.

MakeSFX.exe has been removed from the FreeExtractor install package due to self-imposed size constraints. However, it may be downloaded seperately, for free, from the Disoriented Labs website.
