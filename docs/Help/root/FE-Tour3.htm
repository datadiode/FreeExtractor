Step #2: Options

These options will allow you to change the introduction text that is displayed on the splash screen of your self-extracting archive. While these are optional, it is strongly recommended that these are completed. They give your end users valuable information about the archive's contents.

    Archive Name

    This is the friendly name that FreeExtractor uses to label the self-extractror's titlebar and splash screen. Practically, only the archive name's first 40 characters are displayed, however you may type up to 76 characters here.

    If you do not specify an archive name, it will default to "Unnamed Archive."

    Website Address

    You may embed a website address or email address here, which is especially useful for software distribution. This field is limited to 200 characters. Website addresses must begin with "http://".

    An email address can be specified using the following syntax:

      mailto:user@domain.com

    You may also specify a subject and/or body text in the querystring using the following syntax:

      mailto:user@domain.com?Subject=This is the email subject&Body=This is the email body

    In either case, only the email address is displayed ("user@domain.com").

    Introduction Text

    The introduction text is displayed when a user runs this self extractor. This should contain a short description of the contents of this archive. This is limited to 675 characters in length.
