Authoring Self-Extractors

Two tools are available for creating self-extracting archives: the FreeExtractor Wizard and MakeSFX.exe.

Both generate the same output - 32-bit, graphical Windows self-extracting archives. The difference is that MakeSFX is command line-based, whereas the FreeExtractor Wizard has a graphical user interface.

Most SFX authors should use the FreeExtractor Wizard. MakeSFX was designed specifically for software developers who wish to automate the process of packaging their software (e.g. through build scripts).
